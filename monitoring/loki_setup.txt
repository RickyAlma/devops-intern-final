Monitoring Setup with Grafana Loki

This repository now includes a root-level `docker-compose.yml` file that starts Loki, Promtail, and the demo app.

1. Start Loki and Promtail:

```bash
docker compose up -d
```

2. Verify Promtail is forwarding Docker logs to Loki:

```bash
docker compose logs promtail
```

3. Query logs with Loki's CLI:

```bash
docker run --rm -it --network host grafana/logcli:2.9.1 \
  --addr=http://localhost:3100 \
  query '{job="docker"}' --limit=20
```

Promtail is configured in `monitoring/promtail-config.yml` to scrape Docker JSON logs from `/var/lib/docker/containers/*/*.log` and send them to `http://loki:3100/loki/api/v1/push`.

If your environment uses the older Compose command, you can also run:

```bash
docker-compose up -d
```
